## Summary
This README file describes the purpose and functionality of the PayPal Plugin, a payment integration developed by the Sylius Core Team for the Sylius e-commerce platform. The plugin provides a seamless payment experience by integrating PayPal Commerce Platform, supporting over 100 currencies and various payment methods, including credit/debit cards and regional payment methods. It exists to facilitate secure and efficient online transactions, leveraging PayPal's features such as PCI DSS and 3DS 2.0 compliance, risk management, and dispute resolution.
