# login.feature

**Path**: `features\api\shop\login.feature`

## Summary
This Gherkin feature file defines a test scenario for the API shop login functionality, specifically testing the successful login of a user with a given email and password on a specific channel. The file exists to ensure that the shop's authentication mechanism works correctly, allowing users to access the platform by verifying their credentials. It operates within the context of a multichannel store, where users can log in on different channels, such as "fr" or "de".
