## Summary
The `UpdateOrderApi` class is responsible for updating a PayPal order by sending a request to the PayPal API with the order details, including payment information and shipping address. This class exists to facilitate communication between the Sylius e-commerce platform and the PayPal payment gateway, enabling seamless order updates and payment processing. The class uses various providers and clients to gather and process the necessary data, ensuring accurate and efficient order updates.
