# onboarding.md

**Path**: `plugins\PayPalPlugin\docs\onboarding.md`

## Summary
This documentation file outlines the onboarding process for integrating PayPal with the Sylius e-commerce platform, guiding store owners through the steps required to connect their PayPal business account and enable PayPal as a payment method. The process involves accepting PayPal consents, authenticating with PayPal, and configuring the payment method settings to facilitate a seamless payment experience. The documentation exists to provide a clear and step-by-step guide for merchants to successfully onboard with PayPal and start using the PayPal Commerce Platform integration.
